名单 投稿 说明 GitHub

说明

本站由 freetabris 公开维护。投稿经站长审核后录入。 数据源 data.json 在 GitHub 仓库公开。

投稿渠道

  1. 网页表单：在首页底部提交，自动开 GitHub Issue。最简单。
  2. GitHub Issue：直接用 投稿模板开 issue。
  3. 邮件：发到 lihao10288@gmail.com，主题加「投稿」。
  4. 圈内私聊：微信直接发给站长。

审核标准

  • 必须能拿出**至少一条**佐证：闲鱼差评页 / 微信聊天截图 / 转账记录 / 群证词（多人对账）
  • 纠纷金额至少 200 元（小额建议走平台投诉）
  • 24 个月以内发生的（更早的证据时效不足）
  • 商家是国内圈内卖家（境外的本站不覆盖）
  • 事实陈述，不写主观定性（写「未发货」不写「他就是骗子」）

申诉 / 撤稿

  • 被曝光的商家或任何第三方，任何时候可提交 GitHub Issue 或邮件申诉
  • 申诉需说明：你是谁（不要求实名）+ 你的反证
  • 站长 3-14 天内回复处理结果（撤稿 / 修改 / 维持原内容）
  • 站长对所有内容保留永久撤稿权。提供有效反证立刻撤。

本站不会做的

  • ❌ 不收钱（投稿费 / 删帖费 / 广告 / 捐赠一概不接）
  • ❌ 不撮合交易、不做担保中介
  • ❌ 不为任何被曝光商家做"洗白"承诺
  • ❌ 不向任何第三方主动提供数据

免责

  • 本站对每条记录已尽合理审核义务，但不能保证 100% 真实。投稿人可能伪造证据，双方可能各持证据时站长难以裁判。
  • 本站发布内容仅供圈内人参考，不构成对任何商家信誉的最终评价。
  • 读者基于本站内容做出的交易决策，由读者自行承担后果。
  • 本站不收集读者的浏览数据 / Cookie / 任何识别信息。
