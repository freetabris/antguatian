一、本站性质

本站是一份独立运营的非营利曝光月刊，专注于记录蚂蚁爱好者圈内的交易纠纷事件。

  • 运营主体：独立编辑人，匿名运营
  • 运营性质：非营利（无广告 / 无会员费 / 无抽佣 / 不接受捐赠）
  • 内容来源：用户投稿 + 编辑人人工核实
  • 发布形式：滚动更新，纯静态网站
  • 法律地位：内容发布者（不援引避风港原则）

本站会做的事

  • ✅ 接收符合「投稿须知」四件证据齐全的曝光投稿
  • ✅ 对每篇投稿做人工核实（联系商家、对照证据）
  • ✅ 发布前给被曝光商家 72 小时申诉期
  • ✅ 对所有已发布内容保留永久撤稿权利
  • ✅ 对投稿人和申诉人身份严格保密

本站不会做的事

  • ❌ 不接受任何形式的付费（投稿费、会员费、删帖费、广告费、捐赠）
  • ❌ 不撮合任何交易，不提供担保中介服务
  • ❌ 不开放评论区、不开放注册、不开放表单投稿
  • ❌ 不发布涉及具体物种学名、产地、走私、海关、检疫的内容
  • ❌ 不在国内任何社交媒体宣传
  • ❌ 不向任何第三方主动提供数据
  • ❌ 不向监管机构主动报送任何内容
  • ❌ 不接受任何机构、任何理由要求披露运营者身份

二、投稿须知

本站不是消费维权平台，不撮合交易、不收费、不抽佣、不接受捐赠。 本站发布内容由编辑人匿名运营，对所发布每一篇内容承担编辑责任。 每一篇曝光对编辑人都是法律暴露面，投稿质量直接决定能否发布。

2.1 投诉分两类，门槛不同

类别适用情形显示方式证据要求
商家纠纷（dispute，默认） 货不对版、补发拖延、售后态度差、拒不退款等民事范畴纠纷 脱敏 ID（L***023） 四件标准证据
纯诈骗（fraud） 卷款跑路、收钱不发货且拉黑、冒充等已构成诈骗的情况 明文 ID（wxid_xxx），整个商家档案升级为明文 四件标准证据 + 一件硬证据

fraud 级别的「硬证据」—— 必须任一种：

  • 📄 派出所受案回执（已立案的报案凭证截图）
  • ⚖️ 法院判决书 / 调解书（生效文书）
  • 🚫 监管部门处罚通知（市场监管 / 公安等出具）
  • 💬 平台官方诈骗认定（淘宝 / 闲鱼 / 微信对该 ID 的诈骗判定截图，必须含官方水印）

没有硬证据，再多的聊天截图 + 转账记录都只能定级 dispute。「诈骗」是法律概念，非个人感觉。

2.2 四件必备证据（缺一不发）

  1. 完整聊天记录：从首次接触到纠纷发生的完整对话截图，含对方头像 / ID / 时间戳。不接受片段。涉及你本人真实信息部分自行打码。
  2. 转账记录：能看到金额、时间、对方账户后四位（或 ID）的微信 / 支付宝 / 银行卡转账截图。
  3. 物流单或收货证据：快递单（含单号 / 寄件人 / 收件人）或商家承诺发货的截图或活体到家照片（含外包装）。
  4. 商家联系方式：至少两种以上可对应同一商家（防止冒名）。仅供站方核实，不公开。

2.3 写作规范

✅ 应该这么写：

  • 只描述交易行为：付款 / 发货 / 到货 / 沟通 / 售后
  • 用事实陈述与清晰时间线，客观语气，让证据说话

❌ 不要这么写：

  • 不写交易标的的物种学名 / 产地 / 入境方式
    • ❌ 「我从他那买了一只哥伦比亚的 Paraponera clavata」
    • ✅ 「我向商家 B 购买一只活体，金额 1280 元」
  • 不写侮辱性词汇（骗子 / 垃圾 / 人渣 / 王八蛋）
  • 不写下列敏感词（命中即拒）：走私 / 入境 / 海关 / 检疫 / 入侵物种 / 国家保护 / 三有动物 / 野生动物保护法 / 任何具体物种学名 / 任何原产国地区名
  • 不写承诺性语言（「大家以后都别买他的」「他肯定还会骗下一个」）

2.5 直接退回的投稿

  • 缺四件证据中任意一项
  • 包含敏感词（物种学名、走私、海关等）
  • 用侮辱性词汇而拒绝修改
  • 没有具体金额或时间线
  • 实质内容是「我对他态度不满」而无具体违约行为
  • 商家是境外个人 / 境外平台（本站只覆盖国内圈内商家）
  • 纠纷金额 < 200 元（小额纠纷建议走平台投诉）
  • 24 个月以前发生的纠纷（证据时效不足）

2.6 投稿人保护

  • 站方不会公开投稿人的 ID / 邮箱 / 任何识别信息
  • 商家身份核实环节，站方只发「曝光摘要」给商家，不附原始聊天记录
  • 但站方无法保护你免于被商家自己识别—— 如果纠纷情节足够独特，商家自己可能能猜出是谁。投稿前请自行评估这种风险

三、申诉流程

3.1 谁可以申诉

任何人都可以申诉，包括但不限于：

  • 被曝光商家本人 —— 发布前 72 小时申诉期 / 发布后永久申诉期
  • 被曝光商家的家属 / 朋友 / 同事 —— 可代为提交反证
  • 任何第三方 —— 如发现内容失实、证据造假、记错对象等
  • 投稿人本人 —— 发现自己投稿有错或希望撤回

申诉不需要实名。本站不会要求申诉人提供身份证 / 手机号 / 真实姓名。

3.2 申诉的两个时段

A. 发布前 72 小时（核心保护期）：每篇曝光在正式发布前，本站会主动联系被曝光商家：

站方 → 商家：
  - 收到针对你的曝光投稿一份
  - 投诉摘要（脱敏：不附原始聊天，仅描述纠纷类型/金额/时间线）
  - 询问是否有反证、补充说明
  - 给予 72 小时回复期
  - 不回复视为对投诉内容默认

商家在这 72 小时内可以：提供反证、主动联系投稿人和解（并向站方确认）、要求站方修改不实细节、自认不当并请求不发布。

B. 发布后永久申诉期：正式发布后，被曝光商家或任何第三方仍可随时申诉。本站对所有已发布内容保留永久撤稿权利。

3.3 申诉必须包含的内容

(1) 你是谁（任选其一即可）：

  • 「我是被曝光的商家本人」+ 任一可验证身份（如曝光稿件中提到的手机号 / 微信号尾号）
  • 「我是 XX 朋友 / 家属」+ 你跟当事人的关系说明
  • 「我是第三方」+ 你为何对此事知情

(2) 你的反证（按以下类型至少一种）：

  • 类型 A：交易未发生 —— 证明该笔交易不存在 / 你与投稿人从未联系过
  • 类型 B：交易已正常履行 —— 完整发货物流单 / 收货确认截图 / 售后已完成的聊天
  • 类型 C：投稿存在重大失实 —— 金额 / 时间 / 物品与实际不符（提供对照证据）
  • 类型 D：和解已达成 —— 投稿人撤回 / 双方和解协议截图
  • 类型 E：投稿人身份冒用 —— 真实当事人本人出面声明（站方会联系原投稿人核实）

(3) 你希望的处理结果（任选）：撤稿 / 修改特定细节 / 在原文末尾追加更正声明 / 暂时下线复核（站方会在 14 天内决定终态）

3.4 本站如何处理申诉

申诉到达
   ↓
站方在 48 小时内回信确认收到
   ↓
站方核实身份关联（不需要实名，但要求最低限度关联证明）
   ↓
站方对照你的反证 vs 原投稿证据
   ├─ 反证有效，确有错 → 立即撤稿 / 修改 + 邮件回复
   ├─ 反证部分有效 → 修改细节，保留主体
   ├─ 反证无效或不充分 → 维持原内容，回信说明
   └─ 站方无法判定 → 进入二次核实（最多 14 天，期间标注「申诉中」）

平均 3-14 天。复杂申诉（双方各持证据）可能更久。申诉期间，被申诉内容不会自动下线，除非反证一眼足够强。

3.5 申诉成功后

  • 完全撤稿：原文 URL 返回 HTTP 410 Gone，主动请求搜索引擎清理缓存
  • 修改细节：用 <del> 删除线 + <ins> 插入显示，保留修改痕迹
  • 追加更正声明：原文不动，末尾新增「商家声明」段，体现「双方各执一词」

3.6 申诉无效的情况

  • 「我希望删除，因为这影响我的生意」—— 理由不充分
  • 「我希望删除，否则起诉」—— 本站不接受法律威胁作为撤稿依据；但会认真核实任何有据可查的反证
  • 「投稿人是黑产 / 同行恶意举报」—— 口说无凭；如有证据请提供
  • 滥用申诉（同一商家短期内多次无新证据申诉）
